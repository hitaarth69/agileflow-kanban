# 📋 AgileFlow - Advanced Kanban Board

A full-featured, offline-first Kanban board built with vanilla HTML, CSS, and JavaScript. **This is a level-up project** featuring IndexedDB, drag-and-drop, undo/redo, theming, search, and import/export.

## 🚀 Live Demo
[Click here to try it live!](https://your-username.github.io/agileflow-kanban)

## ✨ Features (Level-Up)
- **📦 IndexedDB** — Persistent, large-scale data storage (beyond LocalStorage).
- **🔄 Drag & Drop** — Move cards between columns and reorder them.
- **↩️ Undo / Redo** — Full history with keyboard shortcuts (Ctrl+Z / Ctrl+Y).
- **🎨 Dark / Light Theme** — CSS Custom Properties for seamless theming.
- **🏷️ Labels & Descriptions** — Rich card metadata with color-coded labels.
- **🔍 Search / Filter** — Debounced search across all cards.
- **💾 Export / Import JSON** — Backup and restore your entire board.
- **📱 Full Responsive** — Works on mobile, tablet, and desktop.
- **⚡ Performance** — Efficient re-renders and debounced operations.

## 🛠️ Built With
- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- IndexedDB API
- HTML5 Drag-and-Drop API
- LocalStorage (theme persistence)
